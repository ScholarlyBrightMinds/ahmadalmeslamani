# Ahmad Z. Al Meslamani

Personal academic site, part of [Scholarly Bright Minds](https://github.com/ScholarlyBrightMinds).

🌐 **Live:** https://scholarlybrightminds.github.io/ahmadalmeslamani/

Auto-updates from Google Scholar (weekly) and Scopus (monthly).

---

## Stack

- Static HTML / CSS / JS (no build step)
- Hosted on GitHub Pages
- Publications fetched by `serpapi_fetcher.py` and `scopus_fetcher.py` via GitHub Actions
- Single source of truth: `theme.config.js`

## Editing

See the [editing guide](https://github.com/ScholarlyBrightMinds/.github/blob/main/EDITING.md) in the org's profile repo for how to update content.
